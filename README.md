# poi-plugin-sortie-checker 出击检查

poi 插件：出击前检查当前舰队在指定海域的达标情况。

## 功能

- **舰队选择**：第1/2/3/4舰队、联合舰队（第1+第2）、游击舰队（第3舰队 7 舰）
- **海域选择**：二级下拉框（海域分组 → 具体图），收录二期全部 37 张通常海域（1-1 ~ 7-5，含 2026-05 新增的 5-6）及 **2026 夏季活动前段 62-1/2/3**
- **出击检查**：出击前提示 **大破舰船**（HP≤25%，出击有轰沉风险）与 **油弹未满**（对照舰船搭载上限）的舰娘名单，全部通过时显示绿色确认
- **输送 TP**：实时显示当前舰队 TP（S胜 / A胜=×0.7 取整）；舰种 TP + 大发系/内火艇/ドラム缶/战斗粮食装备加成（装甲艇·武装大発不计，鬼怒改二 +8）
- **制空值**：每格 `⌊(对空+改修补正)×√搭载 + 熟练度加成⌋` 求和，熟练度按内部经验保底值计算（舰战/水战改修 +0.2×★，爆战 +0.25×★；舰战系熟练加成 [0,0,2,5,9,14,14,22]，水爆 [0,0,1,1,1,3,3,6]）
- **索敌合计**：面板显示值之和
- **33式索敌**：`Cn×Σ[装备系数×(装备索敌+改修系数×√★)] + Σ√素索敌 − ⌈0.4×司令部Lv⌉ + 2×(6−舰数)`，一次显示 Cn=1~4 及自定义系数；联合舰队按 12 格计算
- **6-3 航空侦察分数**：`Σ 水侦/水爆索敌×搭载^0.25 + 大艇索敌×搭载^0.5`（按出击时搭载数，未计道中击坠）
- **索敌 / 航空侦察要求**：各图 33式/航空侦察定量分歧条件（wikiwiki 2026-07 数据），实时显示 满足/随机带/不足
- **制空要求**：各图 Boss 及主要航空战/空袭点的敌制空值（取最强形态），按 确保(×3)/优势(×1.5)/均衡(×2/3)/劣势(×1/3) 换算，实时显示当前舰队制空所处档位及各档所需制空线
- **特殊分歧条件**：各图起点及关键分歧点的舰队构成/航速类路由规则（战舰空母数量、高速统一、特定舰等）
- **推荐阵容**：各图 1~5 个常用编成（攻略/周回/任务用，多血条图按血条区分），含舰种构成、路线与装备要点
- **活动图（62-1/2/3）**：简化展示，只列 **贴条（札）／攻略步骤／路线／推荐阵容（舰种）**，不做定量分歧判定；数据来自 [HKGY/2026SummerEvent](https://github.com/HKGY/2026SummerEvent) 实战攻略，E2 P3 与 E3 大部分编成待检证

## 安装

将本目录复制（或链接）到 poi 插件目录后重启 poi / 重载插件：

```
~/Library/Application Support/poi/plugins/node_modules/poi-plugin-sortie-checker
```

## 注意

- 33式阈值以司令部 Lv120 为基准，低等级时数值判定会偏乐观，建议留余量。
- 标注"存疑"的阈值为 wiki 上带 `?` 的未完全确证数据。
- 敌制空值默认取该点最强形态（削甲/破甲、司令部等级会导致实际偏低，个别图如 4-5 破甲后反而更低），规则文字中已注明弱形态数值。
- 制空判定不含基地航空队与道中击坠的影响。
